# AGENTS.md

## 项目目标
根据 design/stitch/ 中的原型实现“工业企业出海主数据平台”一期官网门户前端。

## 技术要求
- React + TypeScript + Vite
- Tailwind CSS
- 组件化开发
- 桌面端优先，兼顾响应式
- 使用 mock 数据，不接后端

## 代码规范
- 优先复用组件，避免重复代码
- 样式统一管理
- 文件命名清晰
- 不要随意删除已有配置
- 改完后执行构建或检查命令

## 页面范围
- 首页
- 平台介绍 / 政策与补贴
- 企业入驻
- 产品展示
- AI工具

## 输出要求
完成任务后总结：
1. 修改了哪些文件
2. 还缺哪些资源
3. 下一步建议
