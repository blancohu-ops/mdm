# 后端验证清单

## 1. 工程基础

- `backend/` 工程可独立启动
- `pom.xml` 质量插件可执行
- 统一响应和统一异常处理生效
- 请求链路带 `requestId`
- Swagger/OpenAPI 可访问

## 2. 安全

- 未登录访问后台接口返回 `401/403`
- 企业主账号只能访问本企业数据
- 审核员不能执行运营管理员专属动作
- 导出、下载、上传都有权限校验
- 登录、验证码、上传、导出接口有频控
- 密码与验证码不出现在日志中
- 敏感配置不写入仓库

## 3. 业务状态

- 企业状态流转全部符合白名单
- 产品状态流转全部符合白名单
- 企业审核快照与生效资料分离
- 产品审核快照与生效资料分离
- 审核通过/驳回后消息会生成
- 状态变更日志完整可追溯

## 4. 文件与导入

- 图片、PDF、Excel 仅允许白名单格式
- 文件预览和下载走受控接口
- 本地文件目录不会暴露原始路径
- 导入校验结果能精确到行
- 导入失败不会写入脏数据
- 导入完成后消息和状态正确

## 5. 质量基线

- `Spotless` 通过
- `Checkstyle` 通过
- `SpotBugs` 通过
- `JaCoCo` 覆盖率达到门槛
- 新增接口包含参数校验和 OpenAPI 注解
- Controller 没有业务逻辑
- Entity 没有直接暴露到 API 层

## 6. P0 测试集

- `FT-01` 注册成功
- `FT-02` 登录成功/失败
- `FT-03` 企业入驻提交
- `FT-04` 企业审核驳回
- `FT-05` 企业审核通过
- `FT-06` 企业资料变更审核
- `FT-07` 产品草稿保存
- `FT-08` 产品提交审核并上架
- `FT-09` 产品驳回后修改重提
- `FT-10` 产品下架与重新上架申请
- `FT-11` 企业数据隔离
- `FT-12` 角色越权阻断
- `FT-13` Excel 导入成功/失败
- `FT-14` 文件上传/预览/下载权限
- `FT-15` 消息生成与已读未读
